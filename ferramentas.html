O GitHub é uma plataforma usada para versionar e armazenar todos os documentos produzidos pelo grupo. Adicionalmente, o nosso projeto está hospedada na plataforma GitPages.

HTML, CSS E JAVASCRIPT

HTML, CSS e JavaScript são tecnologias amplamente usadas no desenvolvimento de sites e aplicações web.

  • O HTML é uma linguagem de marcação que é usada para criar a estrutura e o conteúdo da página web. Ele define o layout da página, o texto, as imagens e outros elementos visuais.
  • O CSS é uma linguagem de estilo que é usada para adicionar estilo e design à página web. Ele define a cor, fonte, tamanho e posição dos elementos visuais.
  • O JavaScript é uma linguagem de programação que é usada para adicionar interatividade e funcionalidade à página web. Ele permite que você crie animações, interações de usuário e outras funcionalidades dinâmicas na página.

Utilizamos essas tecnologias para a criação das páginas, definição do layout e garantir a responsividade do menu em diferentes dispositivos.

Além disso, implementamos um loader de carregamento para oferecer uma melhor experiência de usuário enquanto o conteúdo do site é carregado.

BootStrap

Bootstrap é um framework front-end de código aberto que fornece uma coleção de ferramentas e componentes para o desenvolvimento rápido e responsivo de sites e aplicativos web. Ele inclui estilos CSS predefinidos, JavaScript e plugins personalizados, além de uma grade flexível para ajudar na criação de layouts responsivos,usamos na criação das tabelas do site.

Whatsapp

WhatsApp é utilizado para aprimorar a comunicação entre os membros do grupo, permitindo que as informações sejam trocadas de forma ágil e eficiente. Através do WhatsApp, são marcadas as reuniões, Isso ajuda a garantir que a equipe esteja sempre bem informada e alinhada, contribuindo para a melhoria da produtividade do grupo.

Google Forms

Utilizamos o Google Forms para coletar dados dos usuários do sistema de forma organizada e eficiente. O Google Forms é uma ferramenta online de criação de formulários que permite a coleta de dados de forma rápida e simples. É possível personalizar os campos do formulário de acordo com as necessidades do projeto, definir perguntas obrigatórias, permitir o envio de arquivos e configurar respostas automáticas de confirmação. Com isso, é possível coletar informações importantes dos usuários de forma padronizada, facilitando a análise e interpretação dos dados coletados.

Vectr

Utilizamos o Vectr para criar a nossa logo de forma simples e eficiente. O Vectr é uma ferramenta de edição vetorial online, que permite a criação de gráficos vetoriais de alta qualidade.

Renderforest

Utilizamos o Renderforest para criar a animação da nossa logo de forma simples e com qualidade profissional. O Renderforest é uma plataforma online de criação de vídeos e animações, que oferece uma variedade de modelos prontos e personalizáveis para diferentes tipos de projetos.
