Apresentação:

Projeto A3 da disciplina de Design de Interfaces, ministrada pelo Professor Fabrício Gabriel Olívio, visa a criação de um projeto educacional que envolve a interface do usuário, seguindo as diretrizes da ONU para o Objetivo de Desenvolvimento Sustentável número 4. Nosso trabalho irá contribuir para a melhorar a experiência do usuário em projetos educacionais, considerando questões fundamentais como acessibilidade, usabilidade e inclusão, e, ao mesmo tempo, promover a conscientização sobre a importância da educação para o desenvolvimento sustentável. Com isso, a gestão e comunicação escolar terá facilidade e otimização de tempo e redução nos custos de matéria-prima e inclusão tecnológica, beneficiando todo seu público-alvo.

Ferramentas

Github

O GitHub é uma plataforma usada para versionamento e armazenamento de código, usamos para hospedar os documentos produzidos pelo grupo e hospedar nosso projeto na plataforma GitPages.

HTML, CSS E JAVASCRIPT

HTML, CSS e JavaScript são tecnologias amplamente usadas no desenvolvimento de sites e aplicações web.

  • O HTML é uma linguagem de marcação que é usada para criar a estrutura e o conteúdo da página web. Ele define o layout da página, o texto, as imagens e outros elementos visuais.
  • O CSS é uma linguagem de estilo que é usada para adicionar estilo e design à página web. Ele define a cor, fonte, tamanho e posição dos elementos visuais.
  • O JavaScript é uma linguagem de programação que é usada para adicionar interatividade e funcionalidade à página web. Ele permite que você crie animações, interações de usuário e outras funcionalidades dinâmicas na página.

Utilizamos essas tecnologias para desenvolver e criar o layout das páginas, garantindo que o site seja responsivo em diferentes dispositivos. Além disso, aprimoramos a experiência do usuário implementando um loader de carregamento com frases variadas.

BootStrap

Bootstrap é um framework front-end de código aberto que fornece uma coleção de ferramentas e componentes para o desenvolvimento rápido e responsivo de sites e aplicativos web. Ele inclui estilos CSS predefinidos, JavaScript e plugins personalizados, além de uma grade flexível para ajudar na criação de layouts responsivos,usamos na criação das tabelas do site e icônes .

Whatsapp

WhatsApp é utilizado para aprimorar a comunicação entre os membros do grupo, permitindo que as informações sejam trocadas de forma ágil e eficiente. Através do WhatsApp, são marcadas as reuniões, Isso ajuda a garantir que a equipe esteja sempre bem informada e alinhada, contribuindo para a melhoria da produtividade do grupo.

Google Forms

Utilizamos o Google Forms para coletar dados dos usuários do sistema de forma organizada e eficiente. O Google Forms é uma ferramenta online de criação de formulários que permite a coleta de dados de forma rápida e simples.

Vectr

Utilizamos o Vectr para criar a nossa logo de forma simples e eficiente. O Vectr é uma ferramenta de edição vetorial online, que permite a criação de gráficos vetoriais de alta qualidade.

Renderforest

Utilizamos o Renderforest para criar a animação da nossa logo de forma simples e com qualidade profissional. O Renderforest é uma plataforma online de criação de vídeos e animações, que oferece uma variedade de modelos prontos e personalizáveis para diferentes tipos de projetos.
